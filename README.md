# Grokking the Coding Interview - Patterns for Coding Questions

This repository contains the course materials for "Grokking the Coding Interview: Patterns for Coding Questions" with enhanced navigation.

## Navigation

Each lesson page includes navigation buttons:
- **Blue "Previous" button**: Navigate to the previous lesson within the same pattern
- **Green "Next" button**: Navigate to the next lesson within the same pattern
- **Violet "Previous Pattern" button**: Navigate to the last lesson of the previous pattern (on first lesson of a pattern)
- **Violet "Next Pattern" button**: Navigate to the first lesson of the next pattern (on last lesson of a pattern)

## GitHub Pages

This repository is configured to be hosted on GitHub Pages. Once pushed to GitHub:

1. Go to repository Settings
2. Navigate to Pages section
3. Select the branch (usually `main` or `master`)
4. Select `/ (root)` as the source
5. Save - your site will be available at `https://[username].github.io/[repository-name]`

## Local Usage

Simply open any HTML file in a web browser. The navigation buttons use relative paths and will work locally.

## Structure

- Each pattern folder contains numbered lesson folders
- Each lesson folder contains language-specific HTML files (typically 4 files: C++, Java, Python, JS)
- Navigation is automatically added to all HTML files
